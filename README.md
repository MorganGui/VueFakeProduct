# loesus

## Project setup

#### Check if node is instaled.
```
node -v
```
#### Install vue and his CLI.
```
npm i vue
npm i -g @vue/cli
```
#### install sass.
```
npm install node-sass sass-loader --save-dev
```
#### Create the projet.
```
vue create [project-name]
```
## Compile the project.
```
cd [project-name]
npm run serve
```
